About Me

I am an American-Israeli software developer and entrepreneur. I grew up in Herzliya, Israel, and moved to the USA with my family in 2003. I started programming at the young age of 11, doing projects that I thought were cool, and soon realized my love for all things technology. As I matured, I decided to pursue a career in the field I loved, and focused on learning as much as I could about programming and software development.

After graduating high school, I studied for a year at the Engineering School of the University of Illinois at Urbana-Champaign, focusing on Computer Science. After studying at UIUC for a year, I began working in Atlanta for various companies, writing software and algorithms as an independent software developer.

As an independent developer, I have gotten a lot of experience working in small, medium, and large businesses, doing any sort of development that the companies need. I specializes in development, efficiency, and breaking things down in order to fix them and make them work better.

I moved back to Israel in 2016 and started working as a Tableau expert, analyst, data scientist, and software developer soon after arriving here. As a Tableau Expert, I learned how to create user-friendly interactive dashboards and visualizations for various types of data sets. As a Data Scientist, I've learned how to create ranks, scores, and visuals to answer business questions from clients.

Selected Projects

Rulebase Classification Algorithm

A classification algorithm for Signals Analytics. The project entailed designing and developing a Natural Language Processing algorithm to take massive amounts of data points, each with its own text description, and then classify what each data point is talking about, based on a set of rules provided.

After the application was built as a monolith, it was then broken it up into decoupled microservices to deploy as highly scalable cloud-based workers. The project is developed primarily in C# and ASP.NET Web API, using ElasticSearch as a search algorithm, and RabbitMQ as a task queue.

WoPair - The Workout Pairer

An application built as a personal endeavor, to allow people to find nearby users with likeminded preferences to workout with. The application utilizes location information and a user database to generate a list of nearby users, with user preferences listed and a match percentage of how closely each user matches the preferences of the current user.

The project was developed in two phases. The backend phase was developed using Elixir and the Phoenix Framework, to allow for maximum performance. The frontend phase is currently in development as a web application, an iOS application, and and Android application.

Tax Automation Process

A series of automation scripts for use by Jamestown, L.P. The scripts automate the process of sending the tax data necessary for the United States Internal Revenue Service to create and retrieve tax returns tax returns from their e-File system.

The project was developed using the Selenium Browser Automation Framework for automating the filing of the returns, along with Ruby scripts, and C# programs (for a nice usable form for Windows computers).
